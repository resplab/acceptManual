4.2 FEV1 Prediction Function

4.2.1 Example

To get started, there is an R data frame with the package of sample patient data:

samplePatients
#> # A tibble: 2 x 7
#>   ID     male   age smoker  FEV1 height weight
#>   <chr> <dbl> <dbl>  <dbl> <dbl>  <dbl>  <dbl>
#> 1 10001     1    70      1   2.5   1.68     65
#> 2 10002     0    42      0   3.6   1.82     84

To get a prediction:

results = predictFEV1(samplePatients)
